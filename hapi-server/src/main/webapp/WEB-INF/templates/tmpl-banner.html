Symphony FHIR Server powered by HAPI FHIR

This FHIR server hosts AI-generated clinical summaries and IPS (International Patient Summary) documents created by Symphony AI. Browse and explore FHIR resources, including IPS-compliant Composition documents with rich clinical narratives.

Features:

  • FHIR R4 compliant server
  • IPS (International Patient Summary) support
  • AI-generated clinical narratives
  • RESTful API with search capabilities
  • Web-based resource browser
